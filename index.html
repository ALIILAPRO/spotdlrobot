How It Works

1

Start the Bot

Open Telegram and search for @spotdlrobot or click the "Try the Bot" button above.

2

Send Music Link

Copy any Spotify, YouTube Music, Deezer or Shazam link and send it to the bot.

3

Download Your Music

The bot will process your request and send you the audio file ready to download.

About SpotDLRobot

SpotDLRobot is an open-source Telegram bot that allows users to download music from Spotify by simply sharing a link. It's built using Python and PHP leverages.

The project is completely free to use and maintained by the open-source community. Contributions and suggestions are always welcome!
